NEW FRONTIER

You land on a frozen planet in a tiny rocket with a robot inside. Harvest and combine materials to discover new materials and elements. Use visual programming to make the robot do the tiresome work for you. Slowly build a gang of self-sustaining robot workers and watch them terraform the planet.

Learn programming and system design basics in a fun way. The game is free playable in web browsers and Chromebooks.
