ABOUT

PLAY WITH MACHINE LEARNING

Learning Factory is relaxing factory building game about the most important thing - making Cats happy. Optimize and automate production chains. Sell crafted items to cats and improve service with Machine Learning. Rebuild the ancient Monument to establish Catopia on Mars.

The game is backed up with real Machine Learning, links to handpicked educational videos, and an in-game wiki to be used as learning material.

  • Renovate a factory on Mars
  • Automate production and sales
  • Collect and analyze tons of data
  • Research, scale, optimize and learn
